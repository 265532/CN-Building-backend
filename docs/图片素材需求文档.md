# 图片素材需求文档

## 一、项目概述

本文档旨在为下一个AI agent提供清晰的中国古建筑智慧学习平台图片素材需求指导，确保所有图片素材符合项目的设计风格和功能需求。

### 项目设计风格

- **主题色调**：棕色系（#4A2E1B、#8C5A2B）、金色点缀（#C6A664）、米白色背景（#F6EFE9）
- **整体风格**：传统中式建筑风格，典雅、古朴
- **图片要求**：高清、真实、符合中国古建筑特色

### 图片格式要求

- **支持格式**：PNG、JPG、WebP
- **推荐尺寸**：
  - 轮播图：800x400像素或更大
  - 专题封面：800x600像素或更大
  - 缩略图：400x300像素
  - 标记图标：64x64像素
- **图片质量**：清晰、无水印、适合多端显示

## 四、图片素材获取建议

### 1.图片来源优先级

1. **自有素材**：优先使用static目录下已有的图片
2. **Unsplash**：高质量免费图片网站，适合获取古建筑摄影
3. **Pexels**：另一个高质量免费图片网站
4. **Wikimedia Commons**：古建筑历史图片较多
5. **自己拍摄/绘制**：如有可能，建议实地拍摄或请专业人士绘制

### 2. 图片处理建议

- 使用图片编辑工具调整尺寸和比例
- 统一图片风格和色调
- 压缩图片文件大小，优化加载速度
- 确保图片清晰度

### 3. 版权注意事项

- 确保图片版权合法
- 优先使用CC0协议或公共领域图片
- 商业使用需注意授权范围

## 五、图片命名规范

为便于管理和维护，请遵循以下命名规范：

### 命名格式

```
[类型]-[具体名称].[格式]
```

### 类型前缀

- `banner-` : 轮播图
- `cover-` : 封面图
- `icon-` : 图标
- `badge-` : 徽章
- `landmark-` : 地标建筑
- `component-` : 建筑构件
- `ui-` : UI元素

### 示例

- `banner-forbidden-city.jpg` (轮播图-故宫)
- `cover-dougong.jpg` (封面图-斗拱)
- `icon-palace-marker.png` (图标-皇宫标记)
- `badge-bronze-residence.png` (徽章-青铜民居)
- `landmark-zhaozhou-bridge.jpg` (地标-赵州桥)
- `component-dougong-detail.jpg` (构件-斗拱细节)
- `ui-back-arrow.png` (UI-返回箭头)
